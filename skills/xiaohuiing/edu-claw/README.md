# EduClaw - English Sentence Practice

An interactive English learning tool for sentence making practice.

## Features

- Sentence practice with smart evaluation
- 3-dimension scoring (creativity, vocabulary, grammar)
- Points system with streak bonuses
- Encouraging feedback

## How to Use

1. Start a conversation with "Learn English"
2. The tutor provides words for practice
3. Make sentences and get feedback
4. Earn points for good performance

## Installation

Copy the skill to your OpenClaw skills directory and restart.

# EduClaw - 英语造句练习

一个互动式英语学习工具，帮助练习英语造句。

## 功能特点

- 造句练习与智能评分
- 三维评分（创意、词汇、语法）
- 积分系统与连续学习奖励
- 鼓励式反馈

## 使用方法

1. 说"学英语"开始对话
2. 系统提供单词进行练习
3. 造句后获得评分和建议
4. 表现优秀可获得积分

## 安装方法

复制到 OpenClaw skills 目录，然后重启。
