# DeadLink
Find broken links before your users do. Scan pages, files, and URL lists.
## Commands
- `check <url>` — Check if a single URL is alive
- `scan <file>` — Scan a file for URLs and check each
- `batch <url_list_file>` — Check URLs from a text file
- `report <file>` — Generate detailed link health report
## Usage Examples
```bash
deadlink check https://example.com/page
deadlink scan README.md
deadlink batch urls.txt
```
