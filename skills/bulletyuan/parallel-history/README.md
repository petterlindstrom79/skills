# 平行历史 - 使用指南

## 快速开始

### 方式1：直接对话触发
直接对AI说以下任意一句话：
- "玩平行历史"
- "我想推演历史"
- "从1917年开始玩"
- "平行历史游戏"

### 方式2：指定历史起点
- "从1840年开始玩平行历史"
- "我想在法国大革命时期建立势力"
- "三国时期我能改变什么历史？"

## 游戏流程

### 第1步：选择历史起点
AI会展示可选的历史节点：
- 🏛️ 公元元年 - 罗马帝国
- ⚔️ 221年 - 三国鼎立
- 🔥 1789年 - 法国大革命
- 💊 1840年 - 鸦片战争
- ⚔️ 1914年 - 第一次世界大战
- 🔴 1917年 - 十月革命

### 第2步：了解背景
AI会详细介绍历史背景：
- 当时的世界局势
- 主要矛盾和冲突
- 各方势力的实力对比

### 第3步：选择阵营
每个起点有3-4个可选阵营：
- 🟢/🔵/⚪/🔴 不同颜色代表不同阵营
- 每个阵营有独特优势和劣势
- 可选择扮演历史人物或虚构角色

### 第4步：政治生命（20年）
- 每回合选择1-5年的时间跨度
- 做出关键决策
- 建立组织或信仰
- 积累政治遗产

### 第5步：观察者模式
政治生命结束后：
- 观看组织演变
- 每回合最多5年
- 持续到遗产磨灭或进入荣耀榜

### 第6步：结局评价
- 🏆 荣耀榜 - 影响深远，名垂青史
- ⚖️ 历史评价 - 褒贬不一，功过相抵
- 🔥 耻辱柱 - 被历史否定和遗忘

## 游戏策略

### 成功要素
1. **建立组织** - 在20年内创建可持续的组织
2. **明确意识形态** - 形成有吸引力的思想体系
3. **培养接班人** - 确保组织能延续下去
4. **适应时代** - 根据局势调整策略

### 避免失败
1. ❌ 过度激进化 - 失去群众支持
2. ❌ 缺乏组织 - 人亡政息
3. ❌ 路线错误 - 与历史潮流相悖
4. ❌ 内部分裂 - 组织从内部瓦解

3. 游戏目的是体验历史选择的影响，而非改变真实历史
