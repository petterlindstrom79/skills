# 📈 A股强势股捕捉器

## 功能

基于 **VCP (Volatility Contraction Pattern)** 放量突破形态，实时筛选强势股。

### VCP 形态特征
- **压缩**：ATR5/ATR20 < 0.5（震幅收窄）
- **放量**：RVOL > 2（成交量/20日均量）
- **涨幅**：3%-8%之间（还没涨停，还有空间）
- **换手率**：> 5%
- **突破**：放量突破前期高点

## 输入

- 市场：A股（默认）
- 形态类型：VCP突破

## 输出

- 推荐股票 3-5 只
- 每只包含：
  - 代码 + 名称
  - 信号类型
  - 符合的因子
  - 所属板块
  - 当前涨幅
  - 强势度 ⭐
  - 止损位 + 目标位
  - 买入理由

## 排除条件

- 已涨停的票（涨幅=10%）
- 科创板/北交所（流动性差）
- 成交额 < 5000万

## 数据源

- 东方财富 / 同花顺
- 备用：新浪财经 / 腾讯财经
