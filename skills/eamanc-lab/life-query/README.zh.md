# Life Query

日常生活查询助手 —— 让 AI 帮你处理生活中的信息查询。

## 当前能力

**快递查询**：输入单号，查快递在哪、几时到。支持顺丰、圆通、京东等主流快递，可自带快递100账号查询。

## 规划中

商品比价、天气查询、生活服务等持续迭代中。

## 快速开始

```bash
bash scripts/run.sh call courier-track --trackingNumber SF1234567890
```

## 安装

```bash
npx skills add eamanc-lab/life-query
```
